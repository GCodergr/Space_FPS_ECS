# Space FPS ECS

Space FPS game using Unity ECS

## Unity Version and Dependencies

- Unity: 2019.4.11f1
- Entities: 0.9.0-preview.6
- Hybrid Rendering: 0.4.1-preview.6
- Unity Physics: 0.3.1-preview
